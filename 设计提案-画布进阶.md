# 设计提案 · 画布进阶（ComfyUI 化）— 待用户勾选

现有底子：每节点独立缓存(nodeBufs) + 按需分发(distribute/accepts) + 常驻输出接口。
新节点类型 = 新物品 + tick 分支 + 复用现有钩子，成本可控。以下按批次提案。

## 第一批（推荐先做，生电收益最大）
1. **过滤器节点**（用户点名）：新物品「过滤器」放入画布，一入一出。
   右键节点开配置：白名单/黑名单模式切换 + 物品多选（复用自动合成机的中英文搜索选择器）。
   放行的走出线，被拦的走默认路由(存储)。上游智能分发会自动只送"可能放行"的，零堵塞。
   合成：铁×4 + 漏斗 + 核心模块 + 红石×2。
2. **节点状态灯**：每个机器节点标题栏色点——绿呼吸=正常产出 / 黄=下游缓存偏高(堵) / 红=缺料停机。
   一眼找瓶颈，纯客户端零开销。
3. ~~连线流动效果~~ ✅ 本轮已做（m71）。

## 第二批（自动化灵魂）
4. **数量传感器节点**：设定「存储里 X 物品 < N 时放行/否则断流」。
   例：铁锭<10000 才让铁农场跑 → 自动补货线，防仓库爆型号。生电味最浓的一个。
5. **开关节点**：手动一键通断一条支线（画布上点击切换），后续可扩红石信号版。

## 第三批（分流精细化）
6. **分配器节点**：多出线按 轮询均分/比例(可设 1:1:2)/溢出式(主满走副) 三种模式分流。
   （现状是顺序注水：先连的先吃饱。）
7. **连线流量显示**：hover 连线显示近 5 秒通过量（需要 per-edge 计数，轻量）。

## 观赏与操作手感（穿插做）
8. **框选多节点拖动 + 分组框(Group)**：ComfyUI 式圈一片机器起个名。
9. **便签节点(Note)**：画布上写备注。
10. **空白右键→搜索添加机器**：直接从背包按名字放入，不用先找格子。
11. **直角走线模式**：工程风连线可选项。
12. **小地图**：节点多了之后的概览导航。

## 不建议做（附理由）
- 合流节点：多入一出现架构天然支持（多台连同一目标即合流），单做节点是空壳。
- 每线独立过滤：过滤器节点已覆盖，且节点比"点线设置"更 ComfyUI、更可视。
