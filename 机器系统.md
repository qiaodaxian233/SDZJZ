## 刷线机（固定产物 = 线）
- **不是空白通用机**：刷线机固定生产「线」——自动击杀蜘蛛并刷取线。
- 需放入结构方块/结构核心才工作；支持多台并发提高产能；支持多种输出与升级。
- 合成配方（照蓝图）：
```
铁锭   观察者   铁锭
红石粉 核心模块 红石粉     → 刷线机 ×1
铜锭   线      铜锭
```
- 说明：后续每种产物做一种对应机器（刷石机/刷铁机…同套路，配方各异）。「核心模块」是共用中间件（需单独配方）。

## 结构方块（结构核心 · 宿主）
- 右键开 GUI（多标签：控制面板/结构管理/性能监控/日志/设置）。
- 刷线机插槽：默认 8 格（可放多台，蓝图示例 4/8 运行中）。
- 核心控制：开机 / 暂停 / 停止。
- 升级模块（提升整组）：**速度 / 数量 / 并发**
  - 速度：缩短周期（铜/金/钻/下界合金；最小周期 1 tick）
  - 数量：+N 产出/次（蓝图示例 等级II = +8/次）
  - 并发：+N 同时运行的刷线机（蓝图示例 等级III = +4）
- 输入缓存（自动补充）：铁锭/铜锭/红石粉/线/核心模块/观察者……
- 输出缓存（示例 64K/64K）。
- 输出模式：**自动输出 / 喷射输出 / 管道优先 / 数据线优先**。

## 输出与传输（两条路）
- 模式 1 管道输出：刷线机 → 物品管道 → 箱子（搬真实物品栈，原版味）。
- 模式 2 数据化输出：刷线机 → 数据线 → 箱子 / 数据面板（只传数据不生成实体，零卡顿、大吞吐；末端数据化转换器变回真实物品）。

## 无电力（作者明确）
- 去除 FE/能量：删掉生产状态的「能量消耗」与底栏的「能量 x/x FE」。
- 底栏能量位替换为**结构完整度**；节流靠 每tick操作预算 + 结构完整度 + 散热（见 SdzjzConfig，configVersion 2 已落）。

## 生产状态字段（蓝图）
当前速度 / 并发数量 / 每秒产出 / 在线机器 / 运行时间 / 结构完整度 / 缓存容量。
