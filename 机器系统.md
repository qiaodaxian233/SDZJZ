# 机器系统（草案 · 待作者确认）

> 对应作者口述：合成刷线机 → 放进结构方块（可多台）→ 插升级（速度/个数/并发）→ 开机出料 → 管道 或 数据线(+数据化转换器) → 箱子/面板。
> ComfyUI 式「连线」落在**世界里**：方块=节点，管道/数据线=连线。不写红石。

## 刷线机（通用生产机）
- 定位：**空白通用生产机**，合成时为空；塞进结构方块后指定「产什么」（选配方/产物）即循环刷取。
- 一种机器刷任何东西（若改为「一台固定刷一种」需作者确认）。
- 合成：
```
铁锭   发射器   铁锭
红石   工作台   红石     → 刷线机 ×1
铁锭   比较器   铁锭
```

## 结构方块（生产核心 · 宿主）
- 右键开简单面板：多个刷线机槽 + 升级槽 + 输出缓存 + 开机按钮 + 输出朝向。
- 可放多台刷线机；点开机直接出料。
- 合成：
```
铁块   玻璃块  铁块
玻璃块 工作台  玻璃块    → 结构方块 ×1
铁块   红石块  铁块
```

## 升级（作者三类 · 对齐设计文档 §7）
- 速度：铜 0.75× / 金 0.5× / 钻 0.25× / 下界合金 0.1×（最小周期 1 tick）
- 个数（每次产几份）：×4 / ×16 / ×64 …
- 并发（一个结构方块同时驱动几台刷线机）：并发模块递增

## 传输
- 管道：搬真实物品栈，原版味、简单、量小够用。
- 数据线 + 数据化转换器：只传数据不生成实体，零卡顿、大吞吐；转换器在末端把数据变回真实物品塞进箱子/面板（可反向把箱子物品数据化上线）。
- 数据化转换器合成：
```
青金石   比较器   青金石
末影珍珠 铁锭     末影珍珠  → 数据化转换器 ×1
青金石   红石     青金石
```
