# 机器路线图（对齐作者 1.21.1 生电大全）

> 关键：结构核心**已通用**——机器槽收任意 MachineItem，运行时按 MachineDef 跑。GUI 一次做好、对所有机器通用（显示 运行状态/机器数/tier/升级）。
> 所以扩机器 = 加「MachineDef + 物品 + 配方」，分下面几桶。

## ✅ A. 单产农场类（免费出，现已支持，drop-in）
已做：刷线机、刷石机、刷骨机、刷火药机、刷腐肉机、刷珍珠机、刷史莱姆机、刷铁机、全自动树场、甘蔗机、竹子农场、刷沙机、刷冰机、黑曜石机。
清单里还能直接加的：紫颂果、仙人掌、下界树场（诡异/绯红菌）、金粒(僵尸猪灵)、恶魂泪、凋灵骷髅(煤/骨)、金粒边境塔——都是「加一条 def」。

## 🔧 B. 多掉落农场（需 MachineDef 小改：加权多产物）
沼泽刷怪塔(14 种掉落)、普通刷怪塔、女巫塔(红石/糖/瓶/发光粉…)、守卫者农场(海晶碎片/砂粒/鱼)、袭击塔(绿宝石/图腾/旗)、溺尸塔(腐肉/铜/三叉戟)。
→ MachineDef 的单 product 升级成 `List<Drop>{item,min,max,chance}`，tick 里逐项按概率产。**下一步就能做。**

## 🍲 C. 消耗输入类（走 consumesInputs 路径，字段已预留）
熔炉组(矿+燃料→锭)、骨粉驱动的高效甘蔗/苔藓机、村民交易所(绿宝石↔物)、猪灵交易所(金↔物)、切石。
→ 结构核心 tick 补「消耗 inputs」分支 + 输入缓存来源。

## 🐛 D. 需要抓物笼子（方向 B，供应"来源生物"）
所有刷怪类的生物来源：蜘蛛/苦力怕/骷髅/僵尸/末影人/史莱姆/女巫/守卫者/劫掠者/凋灵骷髅…
→ 抓物笼子(已有捕获) 接上机器：笼子装的怪决定/驱动对应刷怪机。

## 🏭 E. 规模/多方块（超大工作台 tier 已铺路）
百万刷石、16 核/双维度刷铁、巨型史莱姆(360 万)、2880 超级熔炉组、72k 刷冰。
→ 靠 tier(并发×产量) + 未来多方块扩展位；数值 config 化。

## ⛔ F. 超出"插槽机器"范围（红石工程装置，非机器）
世吞/世界吞噬者、珍珠炮/EIS 发射器、屠龙炮、试炼之地"钥匙农场"里的纯红石计时部分。
→ 这些是红石工程/施工装置，不属于"塞进结构核心的机器"。要做得单独立项，或不做；本模组核心是"量产机器"，不是替代所有红石工程。

## 📌 建议推进顺序
1. B 多掉落（解锁沼泽塔/女巫塔等高价值农场）
2. D 抓物笼子接机器（解锁全部刷怪来源）
3. C 消耗类（熔炉组/交易所）
4. E 规模化数值
