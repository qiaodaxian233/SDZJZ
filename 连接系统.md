# 连接系统（走线）设计 —— 参考 AE

## 现状（第一版，已实现）
- 结构核心产出时，从核心出发做 BFS：
  1. **相邻**（六面）有数据面板/箱子 → 直接存入；
  2. 遇到**数据线**方块则顺着继续路由，把产物送到线路末端连接的数据面板/箱子。
- 于是可以「核心 → 拉一串数据线 → 远处的数据面板/箱子」，不用非得贴着核心。
- 无电力、无通道限制（对齐本模组"去电力"设定），BFS 上限 256 格。

## 与 AE 的对照 / 后续路线
- **AE 网络** = 线缆连出一张网，存储/机器/终端都挂网上，物品走"逻辑网络"而非实体管道。我们的数据面板已是"逻辑存储"，很契合。
- 下一步可做的（按需）：
  1. **多目标分发**：核心把产物分发到网络里"能装下的第一个/多个"面板，而不是只找最近一个。
  2. **共享网络存储**：同一张数据线网络上的多个数据面板合并成一个逻辑库；任意终端可取全网物品（AE 的"ME 网络"感）。
  3. **无线/远程终端**：手持终端查看/取用整张网络。
  4. **细导线模型 + 连接贴图**：把整块数据线换成细导管，六面自动连接（AE 线缆观感）。
  5. **输入路由**：消耗类机器从网络里"取料"，而不只是往外送。
- 这些都不涉及电力/通道，只做连接与逻辑存储聚合。

## 备注
本模组核心是"量产机器 + 逻辑存储"，走线服务于"把产物送进存储/下一台机器"，不追求 AE 那套通道/信道复杂度。

## 存储/连接分期路线（作者定，参考 AE / 简易存储）
- **前期：连线**（已做）——数据线把核心接到数据面板/箱子。参考简易存储那种"一根线连起来"的轻量感。
- **中期：无线(WiFi)**——核心/面板加一个"无线节点"，同一区块/范围内免连线自动组网。
- **后期：卫星连接**——跨区块/跨维度远程组网，全局访问。
- **数据面板绑定多个核心**：一个面板可作为多台结构核心的共同产出终端（聚合）。→ 需从"BFS找最近一个"升级为"面板登记它绑定/连到的核心集合，或核心登记目标面板"。
- **面板随手打开**：手持终端物品，远程打开已绑定的数据面板（后期无线/卫星解锁）。
- 分期解锁：连线(免费) → 无线节点(中期材料) → 卫星(后期材料)。全程无电力。

## 消耗类机器接入方式（已做第一版）
- 消耗类机器 `consumesInputs=true`：结构核心 tick 时 `findPanel` 找到连接的数据面板，
  检查面板里有没有足够输入(×运行台数)，有则扣除、产出按掉落表放回输出缓存→再入库。
- 首台：猪灵交易塔(吃金锭→以物换物战利品)。之后熔炉组/合成机/切石照此加 defConsume 即可。

## 中期：无线(WiFi) —— 已实现
- 新增「无线节点」方块。接在结构核心相邻或其数据线网络上即生效。
- 核心推送产出/消耗机取料时：先走有线(相邻/数据线)；若有线找不到目标且网络上有无线节点，则从"面板登记表"里取**同维度、范围内(config.wirelessRange 默认48格)最近的数据面板**，无线送达/取料。
- 面板登记表：数据面板 tick 时登记自身坐标，markRemoved 时注销（仅服务端、仅已加载面板）。
- 无需两端都放节点：核心侧一个无线节点即可广播到范围内任意面板。
- 配方：铜锭×4 + 末影珍珠×4 + 核心模块。
- 后续：卫星(跨维度/全局)、面板绑定多核心、手持终端。

## 后期：卫星连接 —— 已实现
- 新增「卫星节点」方块。接在核心相邻或数据线网络上即启用。
- 路由链：有线(相邻/数据线) → 无线(48格同维度) → 卫星(全局)。逐级兜底。
- 卫星检索：本维度最近面板(无距离上限)优先；否则遍历登记表里其它已加载维度，取任意数据面板，跨维度送达/取料。
- 配方(后期)：无线节点×3 + 末影之眼×4 + 下界之星 + 核心模块。
- 说明：目标面板所在区块须已加载(登记表只含已加载面板)。后续可做"面板绑定多核心/指定目标"避免多面板时的任意选择。

## 数据面板绑定多核心 / 指定目标 —— 已实现
- 新增「数据链接器」物品：右键数据面板记录目标(pos+维度存物品组件) → 右键结构核心把核心绑定到该面板；潜行右键核心解绑。
- 核心路由优先级：绑定目标(可达) → 有线 → 无线 → 卫星。
- 绑定可达性：同维度需(无线&范围内 / 卫星 / 有线可达 wiredReaches)；跨维度需卫星。
- 多个核心绑同一面板 → 面板聚合多核心产出（面板逻辑存储本就按类型累加）。
- 绑定目标存于核心 NBT(boundPos/boundDim)。

## 手持终端 —— 已实现
- 新增「手持终端」物品：右键数据面板绑定；右键空手/地面远程打开该面板存储界面。
- 原理：服务端 openHandledScreen 用真实面板 BE 建 handler，槽位内容靠同步下发，客户端无需该方块实体 → 跨维度/远处也能显示；取出走服务端真实面板扣数。
- 限制：目标面板所在区块须已加载（getBlockEntity 取得到）。配方：玻璃板×3+末影珍珠×2+核心模块+铁锭×3。
