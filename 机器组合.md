# 机器组合手册（m134 · 照代码实证，不是照感觉写）

> 目标读者：玩家/作者。每条语义都对着 `StructureCoreBlockEntity` 的实际实现核对过；
> 「为什么这样接」写在每张蓝图里。改了路由代码请同步更新本文（架构速查里挂了链接）。

## 一、三层物流，一张图

产出从机器出来后，按优先级走三条路，**永远有兜底、绝不堵死**：

```
机器节点 ──出线──▶ 下游机器（画布连线，进它的输入缓存，按需下发）
   │ 吃不下/没人要
   ▼
定向连线（机器绿口 ↔ 存储节点：定向产出/定向供料，跨维度可用）
   │ 没接定向
   ▼
全局默认路由（贴邻/数据线/无线 连通的存储网络，自动入库）
   │ 全都连不到
   ▼
断网喷射（顶面 2t 一组≈10组/秒，自限；贴上箱子/面板 2 秒内恢复落库）
```

- **出线=按需下发（distribute）**：只把下游机器「吃得下」的物品送下线（accepts 门控），
  吃不下的部分当场走定向存储/默认路由——下游缓存永远不会被垃圾堵死。
- **供料两路**：机器有「存储→自己」的定向供料边就吃边上的；没有就吃全局网络。
  上游机器出线直喂=进本机输入缓存（每种在途上限 BUF_CAP=20 万），优先于两者。
- 万物账本按物品 id 记（普通账本）；**带组件产物（药水/附魔书）只入 m130 精确账本或
  输出缓存，出线一律无视**——id 账本会抹组件，这是保护不是缺陷。

## 二、四种逻辑节点（组合的骨架）

| 节点 | 收料语义 | 转发语义（5t 一拍清运自己的缓存） |
|---|---|---|
| 过滤器 | 白名单放行/黑名单拦截 | 放行的沿出线下发；**拦下的自动回仓**（定向存储/默认路由），不丢不堵 |
| 开关 | 开=收，关=不收 | 开=直通转发；关=持料不动（缓存封顶后上游自然停） |
| 传感器 | 开闸=收 | 监测**仓库里某物品数量**：`<阈值放行` 或 `>阈值放行`；关闸持料不动 |
| 分配器 | 什么都收 | 在所有「吃得下」的下游间**均分**（余数轮转）；分不出去的回仓 |

三条全局规则：
1. **闸门连锁**：一台机器的全部出线目标都是关闸（关的开关/关的传感器/暂停节点）→
   上游整台黄灯暂停，不白产不塞库。
2. **链式拉料（m92/m116）**：任何逻辑节点接了「存储→自己」的供料边，就会替下游从仓库拉料——
   拉的只有下游机器**真实需求**（合成机=目标配方材料、熔炉=可熔炼物、酿造塔=材料+燃料、
   附魔工厂=书+青金石、消耗机=定义输入；支持逻辑节点串联，深度 8 防环），5t 一拍、
  每种在途 4096 上限。**这是把吞吐从「机器自己 40t 拉一次」提到万级/秒的关键**。
3. **暂停即隔离（m110b）**：单节点暂停=不收不产不转发，上游视同关闸。

## 三、产能三轴 + 阶位

- **速度**升级：速率 ×1.5^级，溢出折同 tick 多周期（永不触底，m99）。
- **数量**升级：单周期产出 ×(1+级)。
- **并列**升级：同时运行台数 = 节点台数 ×(1+级) ×阶位倍率。
- **融合（m125/m128）**：画布同类同阶总数≥4 → 节点菜单「融合」聚敛合成高一阶
  （普通→超级→神级→GM），取出/放回阶位不丢。
- 经验：刷怪/熔炼机器按原版经验值攒进**画布核心经验池**，画布领取；附魔工厂从同池扣。

## 四、组合蓝图集（拿来即用）

### 1. 矿→成品直线（下界合金链，m102 打通）
```
深层采掘平台 ──▶ 万能熔炉(残骸→碎片) ──▶ 自动合成机(目标:下界合金锭)
```
为什么：出线按需下发——熔炉只吃可熔炼的残骸/原矿，钻石等其余掉落自动回仓零配置。

### 2. 稳压线（防爆仓/防饥荒）
```
任意高产农场 ──▶ 传感器(监测:该产物, <100000 放行) ──▶ (直通回仓)
```
为什么：传感器监测的是**仓库存量**。仓里到十万就关闸→上游黄灯整台停（闸门连锁），
低于阈值自动续跑。比手动开关省心，服务器负载也稳。

### 3. 分拣线（一产多用）
```
刷怪塔 ──▶ 过滤器(白名单:火药) ──▶ 自动合成机(TNT)
```
为什么：白名单放行火药进合成机，骨头/腐肉等**拦下的自动回仓**——一条线同时供产线和仓库。

### 4. 均分线（一供多耗）
```
全自动农场(甘蔗) ──▶ 分配器 ──┬▶ 自动合成机(纸)
                              └▶ 自动合成机(糖)
```
为什么：分配器在吃得下的下游间均分，余数轮转；某台停机/暂停，份额自动让给其他台。

### 5. 万级吞吐熔炉组（m116 实测姿势）
```
数据面板(供料边)──▶ 过滤器(白名单:远古残骸) ──▶ 万能熔炉×N(高并列)
```
为什么：过滤器接「存储→自己」供料边=链式拉料，5t 一拍替熔炉组从仓库抽料，
残骸碎片从 ~100/秒跳到万级/秒（受熔炉自身产能上限）。

### 6. 经验+附魔全自动闭环（m132）
```
烈焰人塔(攒经验池)         书/青金石(仓库或上游产线)
        │                          │
        └────── 同一画布核心 ──▶ 附魔工厂(目标:锋利V) ──▶ 附魔书精确入库
```
为什么：附魔耗的是**本画布核心的经验池**——刷怪塔必须和附魔工厂在同一核心画布上。
锋利V=书1+青金石15+经验125/本。想给自己留经验：先点「领取」或暂停工厂（同池竞争）。

### 7. 药水线（m131b）
```
地狱疣农场 ─▶┐
甘蔗→糖 产线 ─▶ 酿造塔(目标:强化迅捷·喷溅) ──▶ 药水精确入库
烈焰人塔(燃料粉+强化粉双账) ─▶┘
```
为什么：酿造塔 accepts=计划材料+燃料，上游可连线直喂也可全走仓库；
力量药水的材料烈焰粉与燃料烈焰粉分账不混。

### 8. 跨维度离人基地（m133）
```
主世界：结构核心+机器群(开机=自动强加载)  ⇄  下界：存储核心仓库(端点有期票自动加载)
```
为什么：核心开机就钉住自身区块（重启自恢复），并给存储端点区块续票——人去末地打龙，
主世界产线照跑、下界仓库照收。停机 15 秒全部自动卸载。

### 9. 抓物笼定制刷怪
```
抓物笼(装目标生物,超级工作台合成对应机器) 或 直接放画布产该生物掉落
```
为什么：掉落表跟生物走（MobDrops），凋零骷髅头/潜影壳这类原版难刷的走这条路。

## 五、边界与防坑（都是刻意设计，别当 bug 报）

1. 带组件产物（药水/附魔书/带阶位机器）**出线无视**、机器不吃精确件、合成终端不自动补精确件。
2. 经验不是物品：不走线、不占 accepts；附魔工厂缺经验=红灯（画布经验池数字自查）。
3. 农场类（免费产出）不吃任何供料，给它连供料边没意义。
4. 断网喷射自限≈2 组/秒且会警告一次——这是兜底不是物流方案。
5. 交易所/附魔书直发背包是肌肉记忆保护（m101），现在手动可存仓（m130 精确存储）。
6. 传感器监测的是**存储网络存量**，不是节点缓存、不是产量。
7. 下游全暂停≠上游报错：黄灯=闸门连锁在正常工作。
